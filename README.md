# موقع الأسئلة (صح/خطأ + اختياري)

موقع Flask فيه 202 سؤال (96 صح/خطأ + 106 اختياري) مستوردة تلقائياً من ملفك، مقسّمة على فصول (ذكاء اصطناعي ف1-ف5، وعلم البيانات). أي زائر يدخل، يحل، وياخذ نتيجته فوراً مع شرح للأسئلة اللي فيها ملاحظة توضيحية.

## التشغيل محلياً في Cursor

```bash
cd quiz_site
python -m venv venv
source venv/bin/activate      # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

افتح المتصفح على `http://localhost:5000` — أول مرة تشغّل الموقع بيستورد كل الأسئلة تلقائياً من `questions_data.json`.

## عشان تفتح الموقع من جوالك (خياران)

### الخيار 1: بسرعة، بدون نشر (لو جوالك وجهازك على نفس الواي فاي)

1. شغّل الموقع بـ `python app.py` (فيه أصلاً `host="0.0.0.0"`).
2. لاقي رقم الـ IP المحلي لجهازك:
   - ويندوز: `ipconfig` وشوف "IPv4 Address"
   - ماك/لينكس: `ifconfig` أو `ip a`
   - مثال: `192.168.1.5`
3. من جوالك (على نفس الواي فاي) افتح: `http://192.168.1.5:5000`

ملاحظة: هذا يشتغل بس وقت جهازك مفتوح ومشغّل عليه السيرفر، ولازم يكونون على نفس الشبكة.

### الخيار 2: نشر حقيقي على الإنترنت (يفتح من أي مكان، حتى لو مو نفس الشبكة)

أسهل خيار مجاني: **Render.com**

1. ارفع مجلد `quiz_site` كامل إلى مستودع GitHub جديد.
2. روح [render.com](https://render.com) وسوّي حساب (تقدر تدخل بحساب GitHub).
3. اضغط **New +** → **Web Service** واختر المستودع اللي رفعته.
4. الإعدادات:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
5. اضغط **Create Web Service** — بعد كم دقيقة بيعطيك رابط عام زي:
   `https://your-app-name.onrender.com`
6. هذا الرابط تقدر تفتحه من جوالك أو أي جهاز، من أي مكان.

## هيكل المشروع

```
quiz_site/
├── app.py                  # التطبيق الرئيسي (Flask + SQLite)
├── requirements.txt
├── questions_data.json     # كل الأسئلة (يُستورد تلقائياً أول مرة)
├── quiz.db                 # تُنشأ تلقائياً عند أول تشغيل
├── templates/
│   ├── base.html
│   ├── index.html          # صفحة حل الأسئلة (فلترة بالفصل والنوع)
│   ├── result.html          # صفحة النتيجة + شرح الأخطاء
│   ├── admin_login.html
│   └── admin_panel.html
└── static/
    └── style.css            # تصميم متجاوب يناسب الجوال
```

## ميزات جديدة أضفتها

- فلترة الأسئلة حسب الفصل (ف1 - ف5، علم البيانات) والنوع (صح/خطأ أو اختياري) من القائمة أعلى صفحة الحل.
- بعد التصحيح، لو أخطأت بسؤال فيه ملاحظة توضيحية، بتشوفها تحت إجابتك (💡).
- التصميم متجاوب بالكامل ومناسب لشاشة الجوال (أزرار كبيرة، تمرير سلس).
